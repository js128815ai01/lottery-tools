# WordPress 快速貼上文字版

如果你不想使用 HTML 區塊，可以在 WordPress 編輯器用一般區塊建立以下內容。

```text
好抽工具箱
```

```text
免費線上抽獎、幸運轉盤、數字抽選、隨機分組與機率抽獎工具。
```

## 功能介紹

```text
名單抽獎：貼上姓名或項目名單，快速抽出一位或多位中選者。

幸運轉盤：把獎項或選項放進轉盤，適合直播、教學與現場活動。

數字抽選：設定號碼範圍與排除號碼，自動抽出指定數量。

隨機分組：依組數或每組人數，把名單平均分成多組。

機率抽獎：設定每個獎項的權重或機率，依比例抽出結果。

全頁顯示：抽獎結果可放大全頁顯示，方便投影與現場公布。
```

## 使用流程

```text
1. 點擊「開始使用抽獎工具」。
2. 選擇名單抽獎、幸運轉盤、數字抽選、隨機分組或機率抽獎。
3. 輸入名單、號碼範圍或獎項設定。
4. 按下開始抽選，複製結果或全頁顯示。
```
