## 最短發布步驟

1. 打開 WordPress 後台：

```text
https://wordpress.com/home/toolteam6.wordpress.com
```

2. 進入「頁面」或「網站編輯器」。
3. 編輯首頁。
4. 新增一個「自訂 HTML」區塊。
5. 貼上上面的 WordPress 首頁內容。
6. 按「儲存」或「更新」。
